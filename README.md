# Couch Critic
A web application where one can leave reviews for their favorite movies or shows as well as look up what others had to say about the next thing on your watch list.

## Technologies Used
- HTML5
- CSS3
- React.js
- jQuery
- Postgresql
- Google Fonts
- Ruby on Rails
